#  English Learning Platform

##  Overview
A full-stack web application designed to help users learn English through structured courses, similar to platforms like Coursera.

The platform provides topic-based learning, interactive assessments, and progress tracking. It also includes an admin panel to manage content, topics, and quizzes efficiently.

---

##  Features

###  User Side
- Structured English learning modules  
- Topic-based lessons  
- Interactive quizzes and assessments  
- Progress tracking  

### 🛠 Admin Panel
- Add and manage learning topics  
- Create and manage quizzes/assessments  
- Update course content dynamically  
- Control platform structure
